北理工举办首届校友企业专场招聘会

发布者：zqb  2016/8/6 14:19:00   阅读量：1

分享：


 

2016年4月20日下午，我校在体育馆南厅举办了“北京理工大学校友企业专场招聘会”。漫步者科技股份有限公司、北京国瑞升科技股份有限公司、北京欧亚新科技发展有限公司、北京中航智科技有限公司、秦皇岛康姿百德高新技术开发有限公司、京北方信息技术股份有限公司、宁波理工监测科技股份有限公司等50家校友企业参加了本次招聘会。


 


 

为了给在校学生提供更多的就业机会，同时也为校友企业输送优秀人才，搭建校友与母校、校友与校友之间联络、合作的新平台，招生就业工作处、校友会办公室联合举办了我校首届校友企业专场招聘会。本次招聘会从策划、筹备到组织的各个阶段得到了宣传部、校团委、学生工作处、各学院以及校友企业的大力支持。


 


 

在本次招聘会上，校友企业提供了一大批极具吸引力的工作和实习岗位，校友企业代表与前来求职的学生进行了深入的交流。部分求职者在现场找到了满意的工作或者实习机会。


 


 

从2015年9月10日为2016届毕业生举办首场招聘会至今，我校共举办大中型招聘会34场，小型招聘会341场，累计参会单位1937家，比上一学年增加627家。

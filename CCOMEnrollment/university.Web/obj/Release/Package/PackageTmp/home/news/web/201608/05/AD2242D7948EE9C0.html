软件学院举办“北斗卫星的前身，研发和应用”讲座

2016年8月3日下午三点，北京理工大学大学生软件科技创新创业基地邀请中国科学院光电研究院徐颖研究员作题为“北斗卫星的前身，研发和应用”的专题讲座。陈杰浩老师主持了本次专题讲座，科技创新创业基地暑期集训近200名学生参加了本次专题讲座。

讲座开始前，陈杰浩老师对徐颖研究员进行了详细的介绍。徐颖，北京理工大学校友，中国科学院二代导航专项总体部导航技术研究室副主任，中科院光电研究院建院以来最年轻的研究员、博士生导师。主要研究方向是卫星导航及增强技术。获得中科院北京分院“启明星”优秀人才奖，获首届遥感青年科技人才创新资助计划支持，中国科学院青年创新促进会会员，电子学会高级会员，电子学会青年科学家俱乐部会员。

此次讲座分为北斗前身、北斗建设、北斗应用三大部分。徐颖研究员首先以《史记?天官书》中“斗为帝车，运于中央，临制四乡，分阴阳，建四时，均五行，移节度，定诸记，皆系于斗”的古典授时定位理论为切入点，介绍了北斗卫星导航系统的命名由来，简述了人类导航技术从原始的地文导航、经由天文导航、地磁导航、无线电导航，直至今日发展到人造卫星导航的历史进程。结合自主制造星载原子钟的生动事例，徐颖研究员指出自主研发的卫星导航核心技术在信息化现代战争及社会经济发展等方面十分关键，并进一步详细介绍了我国北斗卫星导航系统从双星快速定位系统，历经北斗一号，北斗二号的建设过程。

正如GPS之父布拉德·帕金森所说“卫星导航应用只受人们想象力的限制”，徐颖研究员还向大家展示了北斗卫星导航系统在维护国家安全、经济安全、人民生命财产安全等方面重要应用，并以中山市北斗产业基地建设为例，介绍了北斗卫星导航系统在抗震救灾、现代农业、气象预测、重点运输过程监控以及居民安保关爱等领域的巨大经济价值与广阔应用前景。讲座最后，徐颖研究员与基地师生分享了研发过程中与研发团队突破难关的心路历程。

 

讲座结束后的交流环节，在场同学们积极向徐颖研究员提出自己对北斗开发工作、北斗军用加密等有关疑问，徐颖也一一耐心细致的解答。


下午四点钟，历时一个多小时的讲座圆满结束。徐颖研究员生动的语言、鲜活的事例、渊博的知识，引发了同学们的强烈反响。此次讲座不仅扩宽了同学们的视野、为大家提供一个和行业精英交流的机会，更为大家带来了深刻的思考。陈杰浩老师最后祝愿基地学子以徐颖研究员为榜样，牢记“胸怀天下，软件报国”，为国家科研工作奉献自己的力量！
